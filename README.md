# OpenCV Fixer

Fix OpenCV Issue for numpy, diffusers, etc. [PyPi](https://pypi.org/project/opencv-fixer/)

## Usage

```bash
# download the autofix tool
pip install opencv-fixer==0.2.5
# execute
python -c "from opencv_fixer import AutoFix; AutoFix()"
```
